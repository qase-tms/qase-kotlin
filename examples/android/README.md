# Android-Kotlin Example

This is a sample project demonstrating how to write and execute Android instrumental tests using Kotlin with Qase TMS integration.

## Prerequisites

Ensure that the following tools are installed on your machine:

1. [Android Studio Hedgehog](https://developer.android.com/studio) (2023.1.1) or newer
2. [JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html) or newer
3. Android SDK (API 24 and above)
4. Android device or emulator to run tests

## Setup Instructions

1. Clone this repository or navigate to the project directory.

2. Open the project in Android Studio and sync Gradle dependencies.

3. Connect an Android device or start an emulator.

4. Once the setup is complete, run the tests by executing:

   ```bash
   ./gradlew connectedAndroidTest
   ```
